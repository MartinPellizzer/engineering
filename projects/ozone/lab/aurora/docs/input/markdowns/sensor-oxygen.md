# Sensore Ossigeno

Si richiede di sviluppare un sensore di ossigeno per il monitoraggio in linea dei concentratori di ossigeno. Tale sensore verrà integrato al sistema di controllo modulare sviluppato in precedenza.

La seguente immagine mostra il sensore di ossigeno attualmente disponibile.

![alt text](projects/ozone/lab/aurora/docs/input/assets/us1010.jpg "Title")

Sarà necessario sviluppare una scheda (modulo) per pemettere la comumicazione tra il sensore di ossigeno e il sistema modulare.

## Stime

- Investimento: 150€ (acquisto schede)
- Tempo Sviluppo: 2 settimane
- Rischio: Basso

## Richiesta a procedere

Si richiede conferma a procedere allo sviluppo della scheda (modulo) del sensore.
