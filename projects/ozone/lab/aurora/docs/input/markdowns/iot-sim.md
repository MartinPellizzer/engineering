# Modulo IoT SIM

Si richiede di sviluppare di una scheda per il controllo remoto utilizzando tecnologia IoT SIM. Tale scheda verrà integrato al sistema di controllo modulare sviluppato in precedenza.

La seguente immagine mostra un esempio di modulo IoT SIM di ozono modulabile per il quale si vuole sviluppare l'integrazione.

![alt text](projects/ozone/lab/aurora/docs/input/assets/iot-sim.jpg "Title")

Sarà necessario sviluppare una scheda (modulo) per pemettere la comumicazione tra il sistema IoT SIM e il sistema modulare.

## Stime

- Investimento: 150€ (acquisto moduli pronti all'uso) + 150€ (acquisto schede)
- Tempo Sviluppo: 2 mesi
- Rischio: Medio
